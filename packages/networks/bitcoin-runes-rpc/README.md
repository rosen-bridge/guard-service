## Introduction

A package to be used as network api provider for @rosen-chains/bitcoin-runes package

## Installation

npm:

```sh
npm i @rosen-chains/bitcoin-runes-rpc
```

yarn:

```sh
yarn add @rosen-chains/bitcoin-runes-rpc
```
